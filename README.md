# Vanilla Tyme — Website

The official website for [Vanilla Tyme](https://vanillatymeservers.com), a whitelisted vanilla-like Minecraft SMP server running since April 2017.

## About

This website serves as the home for the Vanilla Tyme community — featuring server information, player shops, and community resources.

## Tech Stack

- [Nuxt 3](https://nuxt.com/)
- [MongoDB](https://www.mongodb.com/)
- [AWS S3](https://aws.amazon.com/s3/)

## Setup

Install dependencies:

```bash
npm install
```

Copy the example environment file and fill in your values:

```bash
cp .env.example .env
```

Run the development server:

```bash
npm run dev
```
